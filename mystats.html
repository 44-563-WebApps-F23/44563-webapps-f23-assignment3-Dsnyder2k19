Snipper Code: Image Illustration Filter

How do I add a filter to a background image in the body?
Stack Overflow Question 
            :root {
                --color-primary: hsl(0 0% 15%);
                --color-accent: hsl(16.1 5% 55.5%);
                --color-tertiary: aqua;
              }
              
              .painted {
                background: repeating-radial-gradient(
                  circle at center,
                  var(--color-primary),
                  var(--color-accent)
                );
                background-size: 5px;
              }
              
              .painted img {
                filter: blur(4px) contrast(3000%) grayscale(1);
                mix-blend-mode: screen;
                width: 100%;
              }
